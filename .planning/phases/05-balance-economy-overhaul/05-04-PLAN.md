   - `rollMetaLoot('combat', ...)` with `rollMaterialDrops('enemy', enemyId, loopCount, rng, gatheringBoost)`
   - `rollMetaLoot('loop', ...)` with `rollMaterialDrops('terrain', terrainKey, loopCount, rng, gatheringBoost)`
   - `rollMetaLoot('boss', ...)` with `rollMaterialDrops('boss', '', loopCount, rng, gatheringBoost)`

2. Where the LoopRunner accumulates metaLoot on RunState, change to accumulate materials. Find patterns like `runState.economy.metaLoot += ...` or similar and replace with material accumulation:
```typescript
const drops = rollMaterialDrops('terrain', tileType, loopCount);
for (const [mat, amount] of Object.entries(drops)) {
  runState.economy.materials[mat] = (runState.economy.materials[mat] ?? 0) + amount;
}
```

3. Replace the old `loopGrowthOnBossKill` reference with `getLoopGrowth(bossKillCount)` from DifficultyScaler. Import `getLoopGrowth` and `getLoopLength`.

4. For gathering boost: import `getStorehouseEffects` from MetaProgressionSystem. The gathering boost needs the MetaState's storehouse level. If LoopRunner doesn't have access to MetaState, pass `gatheringBoost` as 0 for now and let the scene pass it in. OR add an optional `gatheringBoost` parameter to the relevant LoopRunner methods.

**BossSystem.ts:**

5. Find references to `rollMetaLoot('boss', ...)` and replace with `rollMaterialDrops('boss', '', loopCount)`.

6. Find any `metaLoot` field in boss reward calculations and replace with `materials: Record<string, number>`.

**EventResolver.ts:**

7. Find any event effects that grant `metaLoot` and replace with material grants. Events may have effects like `{ type: "meta_loot", amount: N }` — change to `{ type: "materials", materials: { essence: N } }` or similar.

8. If event effects reference `runState.economy.metaLoot`, change to `runState.economy.materials`.

**TreasureSystem.ts:**

9. If treasure loot includes metaLoot as a possible reward type, update to material rewards.

**MetaPersistence.ts:**

10. Import `migrateMetaState` from MetaState.ts.

11. In the load function (where MetaState is read from IndexedDB), wrap the loaded data with `migrateMetaState(raw)` before returning:
```typescript
const raw = await get('metaState');
if (raw) {
  return migrateMetaState(raw);
}
return createDefaultMetaState();
```

This ensures v1 saves (with `metaLoot: number`) are automatically converted to v2 (with `materials: Record<string, number>` and `storehouse: { level: 0 }`).
