    2. **Create `src/systems/combat/CardResolver.ts`**:
       - `canAfford(card: CardDefinition, state: CombatState): boolean` -- checks stamina >= card.cost.stamina, mana >= card.cost.mana, defense >= card.cost.defense (each if defined)
       - `resolve(card: CardDefinition, state: CombatState, synergyBonus: SynergyDefinition | null): ResolveResult` -- pays costs by mutating state, applies each effect:
         - `damage` to enemy: `Math.max(0, (value * state.heroStrength) - state.enemyDefense)` subtracted from `state.enemyHP`
         - `armor` to self: added to `state.heroDefense`
         - `heal` to self: `state.heroHP = Math.min(state.heroMaxHP, state.heroHP + value)`
         - `stamina` to self: `state.heroStamina = Math.min(state.heroMaxStamina, state.heroStamina + value)`
         - `mana` to self: `state.heroMana = Math.min(state.heroMaxMana, state.heroMana + value)`
         - `debuff` to enemy: reduces `state.enemyDefense` by value
       - If synergyBonus is not null, apply bonus effect (same logic as card effects, bonus applies to trigger card only)
       - Returns `{ totalDamage: number, healed: number, armorGained: number }`

    3. **Create `src/systems/combat/EnemyAI.ts`**:
       - Holds `cooldownTimer: number` initialized from `state.enemyAttackCooldown`
       - `tick(deltaMs: number, state: CombatState, stats: CombatStats): void`
         - Decrements cooldownTimer by deltaMs
         - When <= 0: calculates attack damage based on pattern:
           - `fixed`: `state.enemyDamage`
           - `random`: `state.enemyDamage * (0.8 + rng.next() * 0.4)` (use a simple seeded RNG or Math.random for now)
           - `scaling`: `state.enemyDamage + Math.floor(stats.cardsPlayed * 0.5)` (scales with combat length)
           - `conditional`: if heroHP < heroMaxHP * 0.5 then `state.enemyDamage * 1.5` else `state.enemyDamage`
         - Special effects: `double` = 30% chance to double damage; `stun` = skip hero next card; `debuff` = reduce heroDefense by 3; `lifesteal` = heal enemy by 50% of damage dealt
         - Applies damage: `actualDamage = Math.max(0, damage - state.heroDefense)`, reduces heroDefense first, then heroHP
         - Resets cooldownTimer to `state.enemyAttackCooldown`
         - Emits `combat:enemy-attack` via eventBus with `{ damage: actualDamage, specialEffect }`
         - Increments `stats.damageReceived += actualDamage`

    4. **Create `src/systems/combat/CombatEngine.ts`** (zero Phaser imports):
       - Constructor: `new CombatEngine(state: CombatState)`
       - Creates internal: `SynergySystem`, `CardResolver`, `EnemyAI`, `CombatStats`
       - Fields: `heroCooldownTimer = 0` (starts at 0 so first card plays immediately), `deckPointer = 0`, `lastPlayedCardId: string | null = null`, `isFinished = false`
       - `tick(deltaMs: number): void`:
         1. If `isFinished`, return
         2. Decrement `heroCooldownTimer -= deltaMs`; if <= 0: call `playNextCard()`
         3. Call `enemyAI.tick(deltaMs, state, stats)`
         4. Call `applyPassiveRegen(deltaMs)` -- accumulates time, every 3000ms: +2 stamina (capped at max), +1 mana (capped at max)
         5. Call `checkEndConditions()` -- if enemyHP <= 0: set result='victory', isFinished=true, emit `combat:end`; if heroHP <= 0: set result='defeat', isFinished=true, emit `combat:end`
       - `playNextCard()`:
         1. Loop through deck from deckPointer (max deck.length attempts)
         2. For each card: check canAfford. If yes: check synergy, resolve card, update stats, set heroCooldownTimer = card.cooldown * 1000, emit `combat:card-played`, advance pointer, return
         3. If cannot afford: emit `combat:card-skipped`, advance pointer, increment stats.cardsSkipped, continue loop
         4. If ALL cards unaffordable: set heroCooldownTimer = 500 (retry after 500ms)
       - `advanceDeckPointer()`: deckPointer++; if >= deckOrder.length: reset to 0, stats.reshuffles++, emit `combat:deck-reshuffled`
       - `getStats(): CombatStats` -- returns copy of stats
       - `getState(): CombatState` -- returns reference to state (for UI reads)
       - `isComplete(): boolean` -- returns isFinished
