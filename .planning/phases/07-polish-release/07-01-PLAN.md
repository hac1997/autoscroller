**3. Rewrite `src/audio/AudioManager.ts`:**

- Remove Scene import/dependency from constructor (no longer needed)
- Constructor takes no args. AudioContext created lazily on first `playSFX()` call
- `playSFX(sound: SFXEvent)` — looks up `SOUND_DEFS[sound]`, creates OscillatorNode + GainNode, applies envelope (linearRampToValueAtTime for attack/decay), connects to destination, starts/stops oscillator. Calls `audioCtx.resume()` if state is 'suspended' (autoplay policy).
- Keep `playMusic()` and `stopMusic()` as no-ops (per user decision, no music in v1)
- `setSFXVolume(volume: number)` — clamp 0-1, store as masterVolume
- `setEnabled(enabled: boolean)` — toggles SFX. When disabled, `playSFX` returns early
- `getVolume(): number` and `isEnabled(): boolean` — getters for Settings UI
- Preserve `getAudioManager()` singleton pattern, but remove Scene parameter (no longer needed). For backward compat, accept optional unused arg: `getAudioManager(_scene?: any): AudioManager`
